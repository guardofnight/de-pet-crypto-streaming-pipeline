# Crypto Real-time Streaming Pipeline

Система потоковой передачи данных в реальном времени для анализа криптовалютных сделок.

## Технологический стек

* **Сбор данных:** Python (WebSockets).
* **Брокер сообщений:** Apache Kafka.
* **Хранилище:** ClickHouse (Kafka Engine + Materialized Views).
* **Визуализация:** Grafana.

## Запуск проекта

1. **Запуск инфраструктуры:**
Разверните сервисы с помощью Docker Compose:
```bash
docker-compose up -d

```


2. **Подготовка продюсера:**
Установите необходимые библиотеки (используя `uv` или `pip`):
```bash
uv pip install kafka-python-ng websocket-client

```


3. **Запуск сбора данных:**
Запустите скрипт продюсера:
```bash
uv run producer.py

```


4. **Настройка ClickHouse:**
Выполните SQL-запросы для создания таблиц для автоматического наполнения данных.
5. **Визуализация:**
В Grafana добавьте источник данных ClickHouse (протокол HTTP, порт 8123) и создайте дашборды для мониторинга цен и объемов торгов.

## Основные компоненты

* `producer.py`: скрипт подписки на WebSocket API биржи и отправки данных в Kafka.
* `docker-compose.yaml`: оркестрация всех сервисов.
* `clickhouse`: обработка входящего потока данных через движок Kafka Engine.
